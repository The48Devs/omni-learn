Gamification logic

Organization -> Course(s) -> Module(s) -> Activities

Courses,
    Each course has multiple modules
    Each module has multiple Activities
    Activity point total is shown with the Module
    Module points total is shown inside the course leaderboard and completion

Activity completion Point Criteria, (Total 30pts)
    Accuracy [%/10]
    Speed [time allocated % (100 -> 0) /10]
    B4 Deadline (If any) [10] 
    
    
Levels,
    Each organization has a seperate level per student
    1/4 of all points gained and rounded upto 5 get added as xp
    level xp req goes up by +50% every level and rounded upto 5
    Lvl(0) = 5xp
    The level/xp box on the top of student porrtal show the last active organizaton
    every student that joins automatically gets added to the organization leaderboard at level 0
    
Streak,
    Streak is universal
    1 activity per day for the streak to be counted.
    a update streak function needs to be called at the completion of every activity. (Different criteria for different Activities)
    The last streak date for the student is compared against the current date to check if the streak has been counted or the day.
    If a day is missed the streak is reset to 0.

leaderboards, (Streak only shows students that are part of the organization/Course)
        Organization[Level/Streak]
        Course[Points(Activities are combined)/Streak]
        Module(points of all Activities inside)
        Activity[Points + completion time]
